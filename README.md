# Vet Dog App 🐾

**Онлайн магазин ветеринарных товаров** — учебный проект, представляющий собой полноценное веб-приложение с современным интерфейсом и надежной серверной частью.

## О проекте

Vet Dog App — это удобный онлайн-ветеринарный магазин, созданный для отработки навыков разработки fullstack-приложений. Проект демонстрирует взаимодействие клиентской части на React и серверной на Java Spring с использованием JWT-аутентификации и Hibernate для работы с базой данных.

## Технологический стек

### Backend
- **Java Spring** — основной фреймворк для построения серверной логики
- **Spring Security + JWT** — реализация безопасной аутентификации и авторизации
- **Hibernate** — ORM для удобной работы с базой данных
- **MySQL** — реляционная база данных для хранения информации

### Frontend
- **React** — библиотека для построения пользовательского интерфейса
- **Vite** — современный инструмент сборки для быстрой разработки

## Ключевые возможности

- ✅ Регистрация и авторизация пользователей с использованием **JWT-токенов**
- ✅ Каталог ветеринарных товаров с фильтрацией по категориям
- ✅ Личный кабинет пользователя
- ✅ Корзина покупок
- ✅ Административная панель для управления товарами

## Основные технологии в деталях

### Backend (Java Spring + Hibernate)
Проект демонстрирует уверенное владение современным стеком Java-разработки:
- Построение REST API с помощью Spring Boot
- Настройка безопасности через Spring Security с JWT-аутентификацией
- Работа с базой данных через Hibernate (JPA)
- Связь между таблицами (One-to-Many, Many-to-One)

### Frontend (React + Vite)
Клиентская часть реализована на React с использованием функциональных компонентов и хуков, что обеспечивает быстрый отклик интерфейса и удобство разработки.
